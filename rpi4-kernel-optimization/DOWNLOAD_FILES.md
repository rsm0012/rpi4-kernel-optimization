# Download These Files

Before running the installation, download these files to your working directory:

1. **stage3-arm64-openrc-20251116T233105Z.tar.xz** (221 MB)
   https://distfiles.gentoo.org/releases/arm64/autobuilds/20251116T233105Z/stage3-arm64-openrc-20251116T233105Z.tar.xz

2. **gentoo-20251114.tar.xz** (46 MB)
   https://distfiles.gentoo.org/snapshots/gentoo-20251114.tar.xz

Place both files in the same directory as gentoo_env.sh before proceeding with installation.
